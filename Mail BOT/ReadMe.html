Mail Sending Bot Using ClointFusion

    • Mail Supported

      • Gmail
      • Outlook

    • Automatic Mode

      • Remembers your Previous Inputs and Completes the BOT

    • Semi Automatic Mode

      • Remembers and asks for confirmation and give chance to edit.

    • Details Required

      • To Address

        • Can write multiple email addresses separated by ',' (comma)

      • Subject

        • Subject can be written in a single line. Characters Support (UTF-8)

      • Body

        • Body is a multi line content. Characters Support (UTF-8)

      • Attachments

        • Supports multiple attachments, just browse the files you want to attach.
